课程总览CUDA Foundation Course

Lesson 02 · 最小 CUDA 程序

CUDA 编程入门

第二节承接第一节的概念入口，正式走进一段最小 vector add 程序。核心问题是：CPU 怎么准备数据，数据怎么送到 GPU，GPU 上哪些线程算，结果怎么回到 CPU。

这一节为什么这样简化

这一节只围绕 add_cpu.cpp 和 add_cuda.cu 两条主线，不把 CUDA 讲成 API 字典。每个新词都服务同一个流程：Host 准备、H2D、kernel、D2H、释放和验证。

PPT 故事线

  1. 先从第一节概念进入代码目标。
  2. 用 Host/Device 和内存图画清楚运行场景。
  3. 把 CPU for-loop 迁移成 CUDA 五步结构。
  4. 重点解释 Thread/Block/Grid、launch 配置和索引公式。
  5. 最后用编译、运行、Nsys 和性能入口衔接第三节。
