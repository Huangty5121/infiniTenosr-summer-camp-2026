Course Overview

CUDA 并行编程课程发布包

这个目录是给导师和后续协作者看的干净发布包：每节课用一个 HTML 大纲对照 PPT 逐页说明讲解内容、设计意图、可视化作用、讲课提示和容易误解的边界。

课程数量

4 节基础课

PPT 页数

150 页，分别为 28 / 30 / 40 / 52

大纲形式

只保留 HTML 大纲，不再以 LaTeX/PDF 大纲作为发布入口。

课程文件

Lesson 01 · 并行编程与 CUDA 入门28 slides · outline.html + slides.pptx Lesson 02 · CUDA 编程入门30 slides · outline.html + slides.pptx Lesson 03 · CUDA 性能模型导论40 slides · outline.html + slides.pptx Lesson 04 · float、向量化访存与 half52 slides · outline.html + slides.pptx

总故事线

  1. Lesson 01 建立并行、CPU/GPU 分工和 CUDA 入口。
  2. Lesson 02 用最小 vector add 程序跑通 Host/Device、线程层级、编译和运行验证。
  3. Lesson 03 从能跑进入性能诊断，建立 byte、FLOP、AI、Roofline 和 NCU 证据口径。
  4. Lesson 04 把诊断转成优化动作，围绕向量化访存、alignment、float3、half/half2 和实验验证展开。

Lesson 04 额外包含 show-float.html，用于课堂演示向量化访存和 alignment 的简化动画。
