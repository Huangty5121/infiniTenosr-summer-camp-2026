# 大模型推理服务系统

这个目录是面向 GitHub 发布和导师查看的干净课程包，由原始 `AI 模型导论与推理基础原理` 拆成两讲。

## 课程

- `第一讲/`: 从智能到感知器，30 slides。
- `第二讲/`: 从多层感知器到训练与推理，32 slides。

每讲目录保留：

- `outline.html`: 逐页讲课大纲、页面设计目的、讲授话术和课堂边界。
- `slides.pptx`: 对应课堂 PPT。
- `slide-previews/`: HTML 大纲中使用的每页 PPT 预览图。
- 第二讲额外包含 3 个课堂 HTML 演示：`mlp-xor-demo.html`、`mnist-vector-demo.html`、`matrix-parallel-demo.html`。

## 来源

- 主来源：两份 DOCX 课程文档。
- 参考核对：原始 32 页 PDF。
- 不放入发布包：291MB 原始视频、DOCX 源文件、PDF 源文件、临时截图和生成过程文件。

## 视觉资产

本包只在确实有讲解价值的位置使用 imagegen 生成教学配图：

- 第一讲 P02：四类智能例子图，用来开场比较输入、输出和中间处理方式。
- 第一讲 P06：自然智能三联图，用来打破“智能一定等于人脑”的直觉。
- 第一讲 P12：专家系统流程图，用来说明规则库和推理机，而不是神经网络训练。
- 第一讲 P23：感知器结构图，用来把输入、权重、激活和输出串成判断流水线。
- 第二讲 P23：贪吃蛇强化学习图，用来解释状态、动作、奖励。
- 第二讲 P25：围棋策略/价值图，用来讲 AlphaGo 的策略与价值直觉。
- 第二讲 P28：残差和归一化图，用来讲深层网络稳定训练的工程直觉。
- 第二讲 P31：推理加速硬件图，用来承接矩阵乘、并行吞吐和内存访问。

其他概念优先用可控的 PPT 原生图形或 HTML/CSS/canvas 动画解释。

## 讲授边界

这两讲是给零基础或弱基础学生建立 AI 模型和推理系统入口。PPT 不承担完整逐字稿，详细讲法写在 HTML 大纲和 PPT speaker notes 里。涉及 AlphaGo、特殊网络结构、智能芯片等内容时，按“建立直觉和后续课程衔接”处理，不把简化说法当作严格证明。
